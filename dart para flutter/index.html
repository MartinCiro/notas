Condiciones

  • Para las condiciones tenemos el if que es una estructura comparativa, evaluara si la condicion es verdadera o falsa y con respecto al resultado hará una operacion, su estructura es simple:
    Ejemplo:

    if (condicion) {
    operacion;
    } else if (condicion) {
    Operacion;
    } else {
    Operacion al ser falso;
    }
    }

  • El Switch evalua los parametros y con respecto a ellos da un resultado especifico
    Ejemplo:

    Switch(valorPasar) {
    case 'encasodeser':
    Operacion;
    break;
    case 'encasodeser':
    Operacion;
    break;
    default:
    Operacion;
    }
    }

Ciclos

Existen 3 tipos de ciclos:

  • El For es un ciclo en donde el primer valor es el desde donde empiezo, impone una condicion y un incremente
    Ejemplo:

    for (InicioDeLaVariable; comparacion; Incremete/Decremento) {
    Operacion;
    }

  • El While es la union del For y del if, primero solicita una operacionm una Condicion y si es verdadera repetira la operacion
    Ejemplo:

    While (condicion) {
    Operacion
    }

  • El Do While es similar al while, la diferencia es que este ejeucutará por lo menos una vez
    Ejemplo:

    do {
    operacion;
    Incremento/Decremento;
    } while(condicion);
